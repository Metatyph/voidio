What is Void.Io?

Void.io is an open source artificial intelligence that is capable of Deep Learning. What it learns from contributors and users alike, it uses in turn to help with coding.

It is designed with C# and Javascript in mind, with extensive support for Unity and Unreal Engine. When active, it is capable of autmatically correcting errorneous code, as well as generating whole blocks of code to fit any situation.

Void.io uses Electron for its backend, and thus plays nice with Atom and any other compatible text editors.
